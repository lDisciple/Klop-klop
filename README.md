# Klop-klop
The South African network mapper.

## Background
A simple and amusing lunch break project aimed at decolonizing network mappers.

## Usage
```bash
    python klop-klop.py [-h] [--min-port MIN_PORT] [--max-port MAX_PORT] [--threads THREADS] [--no-color] ip
```
